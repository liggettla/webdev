5.2 Pickles

Writing data in pickle format

import pickle
p = open('principle.pkl', 'wb')
pickle.dump(principleData, p)
p.close()

Reading data in pickle format

import pickle
p = open('principle.pkl', 'rb')
principleData = pickle.load(p)
p.close()
